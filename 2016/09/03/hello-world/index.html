Run server

表明Hexo Server已经启动了，在浏览器中打开http://localhost:4000/，这时应该可以看到Hexo已经为你生成了一篇blog。你可以按Ctrl+C停止Server。

Create a new post

hexo new “My New Post”会在source_posts目录下生成一个markdown文件：My-New-Post.md.
我们可以使用一个支持markdown语法的编辑器（比如Sublime）来编辑该文件。
