# 文章封面

文章封面的意思就是：在博客首页的时候会显示文章的封面图片，点击这篇文章的标题，进入文章后，不显示这张图片。

如果想添加文章封面的话，在写文章的时候，需要添加一个字段属性：`summary_img` 。

<!-- more -->

例如：
```
---
title: CSS 各种Hack手段
date: 2017-06-25 03:25:24
categories: 前端
tags: [CSS]
comments: false
summary_img: /images/css-hack-1.png
---
```
